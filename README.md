## 📖 Overview

**Leviathan's Rest** is a browser-based 3D exploration game where players descend into the remains of an ancient sea titan. Inside its enormous skeleton lies a forgotten civilization of glowing coral cities, abandoned laboratories, mysterious ruins, and unique marine ecosystems waiting to be explored.

Your mission is to recover lost relics, scan marine life, collect hidden pearls, manage your oxygen supply, and uncover the mystery behind the legendary Leviathan.

The game focuses on exploration, atmosphere, and environmental storytelling rather than combat.
if you want to play it directly [leviathans-rest.html](https://github.com/user-attachments/files/29687956/leviathans-rest.html)

## 🎮 Gameplay

Explore a massive underwater world while completing objectives:

- 🦴 Recover all **5 Ancient Relics**
- 💎 Collect **12 Abyssal Pearls**
- 📚 Scan marine creatures and ancient structures
- 🌊 Surface to replenish oxygen
- ♨️ Find thermal vents to survive longer underwater
- ❤️ Restore the Leviathan's Heart

## 🚀 Running the Project

Clone the repository

```bash
git clone https://github.com/yourusername/Leviathans-Rest.git
```

Open the project folder

```bash
cd Leviathans-Rest
```

Start a local server

Example using Python:

```bash
python -m http.server
```

or

```bash
npx serve
```

Then open

```
http://localhost:8000
```

## ⭐ Why This Project?

The goal of **Leviathan's Rest** was to explore what can be achieved using only browser technologies.

Without relying on traditional game engines like Unity or Unreal Engine, the project showcases how modern WebGL and Three.js can deliver immersive 3D experiences directly in the browser.
